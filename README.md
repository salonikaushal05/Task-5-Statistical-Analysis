#  Task 5: Statistical Analysis of Data

This project was completed as part of the Data Analytics Internship program. The objective was to perform statistical analysis on a student academic performance dataset using Python and statistical libraries.

The analysis includes descriptive statistics, hypothesis testing, p-value calculation, and T-test implementation to validate data quantitatively.

To analyze student academic performance data and apply statistical techniques to gain meaningful insights.

## Dataset

**Dataset Name:** Student Academic Performance & Attendance Dataset

---

##  Analysis Performed

### 1. Data Exploration
- Loaded and inspected the dataset
- Checked dataset structure and dimensions
- Identified missing values

### 2. Descriptive Statistics
Calculated:
- Mean
- Median
- Mode
- Standard Deviation
- Minimum and Maximum Values

### 3. Hypothesis Testing
Performed an Independent Sample T-Test to determine whether attendance significantly affects student performance.

---

##  Results

### Descriptive Statistics (Math Score)

| Metric | Value |
|----------|----------|
| Mean | 75.20 |
| Median | 75.26 |
| Mode | 105.38 |

### T-Test Results

| Metric | Value |
|----------|----------|
| T-Statistic | -0.4265 |
| P-Value | 0.6698 |

---

##  Interpretation

The p-value obtained from the T-Test is **0.6698**, which is greater than the significance level of **0.05**.

Therefore, the **Null Hypothesis (H₀)** cannot be rejected.

This indicates that there is **no statistically significant difference** in Math Scores between students with high attendance and low attendance.
